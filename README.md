# 🏫 School Information System (Sistem Informasi Sekolah)

Sistem Informasi Sekolah adalah aplikasi berbasis web yang berfungsi sebagai portal informasi digital bagi siswa, guru, dan masyarakat luas. Aplikasi ini memungkinkan pihak sekolah untuk mengelola konten secara dinamis melalui dashboard admin yang aman.

## 🌟 Fitur Utama

- **Public News Portal:** Menampilkan pengumuman, berita terkini, dan profil sekolah secara interaktif.
- **Dynamic Content Management:** Admin dapat mengelola konten berita, data guru, dan fasilitas sekolah tanpa menyentuh kode.
- **Advanced Security:** Implementasi sistem login admin menggunakan **Bcrypt Password Hashing** (`password_verify`) untuk mencegah kebocoran data.
- **Responsive Interface:** Desain yang menyesuaikan dengan berbagai perangkat (Mobile & Desktop).

## 🛠️ Tech Stack

- **Backend:** PHP Native (OOP Implementation)
- **Database:** MySQL
- **Frontend:** HTML5, CSS3, JavaScript
- **Security:** Password Hashing (Bcrypt), SQL Injection Prevention (Prepared Statements).
