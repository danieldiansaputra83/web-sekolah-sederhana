# web-sekolah-sederhana
Sistem Informasi Sekolah adalah platform manajemen konten (CMS) berbasis web yang dirancang untuk mendigitalisasi informasi dan administrasi sekolah. Proyek ini mencakup fitur pengelolaan berita, data guru, serta panel admin yang aman menggunakan standar enkripsi Bcrypt untuk autentikasi pengguna.
